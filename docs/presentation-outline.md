# Demo Presentation Outline

## Slide 1: Problem in One Line

Incident response is slow because teams forget solved failures under pressure.

## Slide 2: Product in One Line

Flashback Ops turns past outages into ranked, memory-driven incident plans with visible confidence deltas.

## Slide 3: What Makes It Different

- Baseline no-memory plan and memory-augmented plan are shown together
- Memory quality is explicit through scoring and recall evidence
- Feedback closes the loop and improves future performance

## Slide 4: Live Walkthrough

1. Seed memory
2. Trigger incident query
3. Compare baseline vs memory plan
4. Show recalled incidents and root causes
5. Submit feedback and close learning loop

## Slide 5: Results

- Scenario memory boost range: +0.46 to +0.49
- Average memory boost: +0.477
- Fast operational story for judges in under 3 minutes
